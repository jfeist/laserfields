Detailed Description

Return the fourier transform \( \tilde A(\omega) = 1/\sqrt{2\pi} \int \exp(-i\omega t) A(t) \mathrm{d}t = i\tilde E(\omega)/\omega\). if called with a type(laserfield) argument, gets \(\tilde A(\omega)\) for just that one field, otherwise gets sum of all fields in all_laserfields.
